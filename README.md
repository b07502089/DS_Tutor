# Flask api
**Usage**
```bash
# Enable backend venv
source api/bin/activate 

# Set Flask variables
export FLASK_APP=main.py

# Run Flask Backend
flask run --reload --debugger

# --- It will open server on http://127.0.0.1:5000/ ----

#Unable venv
deactivate
```
